# 🏗️ NeuroPept AI 系統架構

## 四層架構

### Layer 1：用戶介面（Frontend）
- Streamlit Web App
- 對話視窗 + Dashboard + 3D 視覺化

### Layer 2：GenAI 核心（Brain）⭐
- MaiAgent / GPT-4 Agent
- 需求理解、任務拆解、工具調用、多輪對話

### Layer 3：工具層（Tools）
1. ProtGPT2 序列生成（GenAI）
2. PLM 活性預測
3. 性質評估（毒性/穩定性/滲透性）
4. ESMFold 3D 結構
5. 競品比對
6. GPT-4 報告生成（GenAI）

### Layer 4：資料層（Data）
- 神經胜肽 DB（Pos 1000 + Neg 1000）
- 競品 DB（Argireline 等 20 種）
- 文獻 RAG（PubMed）
- 對話歷史（SQLite）
